HOW TO USE THIS PROJECT
================================================================================

1. IMPORT INTO ANDROID STUDIO:
   - File > Open > Select bizconnect-v2 directory
   - Wait for Gradle sync
   - Build should complete successfully

2. RUN ANDROID APP:
   - Click Run > Run 'app'
   - Select connected device or emulator
   - App will launch

3. BUILD RELEASE APK:
   - Build > Build Bundle(s) / APK(s) > Build APK(s)
   - APK will be at: app/build/outputs/apk/release/

4. RUN BACKEND SERVER:
   - Method A (Docker): docker-compose up -d
   - Method B (Local): ./gradlew :server:run
   - API available at: http://localhost:8080
   - Health check: curl http://localhost:8080/health

5. SET AS DEFAULT SMS APP:
   - Install APK on device
   - Open Settings > Apps
   - Find BizConnect > Set as default messaging app
   - Grant all requested permissions

ANDROID MANIFEST CRITICAL SECTIONS
================================================================================

The manifest includes 4 REQUIRED components for default SMS app status:

1. MainActivity - MAIN launcher activity with SMS intent filters
   <action android:name="android.intent.action.SEND" />
   <action android:name="android.intent.action.SENDTO" />
   <data android:scheme="sms|smsto|mms|mmsto" />

2. SmsReceiver - Receives incoming SMS
   <action android:name="android.provider.Telephony.SMS_DELIVER" />
   priority="999"

3. MmsReceiver - Receives incoming MMS
   <action android:name="android.provider.Telephony.WAP_PUSH_DELIVER" />
   data mimeType="application/vnd.wap.mms-message"

4. HeadlessSmsSendService - Quick reply from lock screen
   <action android:name="android.intent.action.RESPOND_VIA_MESSAGE" />
   permission="android.permission.SEND_RESPOND_VIA_MESSAGE"

ALL COMPONENTS ARE PROPERLY CONFIGURED AND EXPORTED.
