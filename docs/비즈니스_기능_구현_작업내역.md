### 수정 파일
| 파일 | 작업 |
|------|------|
| `data/preferences/AppPreferences.kt` | DataStore 기반 전면 재작성. 과금/한도/구독/엔진 설정 추가 |
| `domain/engine/QueueEngine.kt` | 내부 중복 TaskEntity/TaskDao 제거, DataEntities.kt로 이동 |
| `domain/engine/ScheduleEngine.kt` | 내부 중복 ScheduledMessageEntity/ScheduledMessageDao 제거 |
| `domain/engine/DataEntities.kt` | 엔진 도메인 레벨 데이터 클래스 통합 관리 |
| `data/local/db/dao/TaskDao.kt` | getPendingTasksLimited, updateAllPendingStatus 메서드 추가 |

### AppPreferences 새 필드
```
getDailyLimitCount() / setDailyLimitCount()     — 무료 일일 한도 (기본 50)
getPaidTierDailyLimit() / setPaidTierDailyLimit() — 유료 일일 한도 (기본 150)
getSubscriptionTier() / setSubscriptionTier()    — 구독 등급 (free/paid)
getCreditBalance() / setCreditBalance()          — 충전 잔액 (기본 0.0)
getSmsCost() / setSmsCost()                      — 단문 단가 (기본 9.8원)
getLmsCost() / setLmsCost()                      — 장문 단가 (기본 29.0원)
getMmsCost() / setMmsCost()                      — MMS 단가 (기본 63.0원)
getQueueThrottleMs() / setQueueThrottleMs()      — 큐 쓰로틀 (기본 500ms)
getLimitMode() / setLimitMode()                  — 한도 모드 (기본 "safe")
getOverseasBlockEnabled() / setOverseasBlockEnabled() — 해외번호 차단
```

### 수신자 선택 4가지 방법
1. **전체 주소록** — 체크박스 리스트 + 전체선택/해제
2. **카테고리별** — FilterChip으로 카테고리 멀티 선택
3. **이름 검색** — contactDao.searchSync() 실시간 검색
4. **번호 직접 입력** — 전화번호 입력 + InputChip

### 발송 흐름
```
수신자 선택 → 템플릿/직접 입력 → 이미지 첨부(선택)
→ 변수 자동 치환 (%이름% → 실제 이름)
→ BulkSendConfirmDialog (수량/비용/한도 확인)
→ 발송 → 진행률 표시 → 완료
```

### 수정 파일
- `BulkSendViewModel.kt` — @HiltViewModel 완전 재작성 (SmsSender 연동, 변수 치환, 비용 계산)

## 이전 세션 작업 (기본 메시지 기능)

### SMS 수신 최적화
- SmsReceiver: 스팸체크/threadId/연락처/설정 4가지 **병렬 async** 실행
- 알림을 먼저 표시 → DB 저장은 후순위

### MMS/LMS 장문 발송 (핵심)
- `sendMultimediaMessage()` API 사용 금지 (삼성 이중발송 문제)
- **MMSC 직접 HTTP POST** 방식 (Textra/QKSMS와 동일)
- 한국 통신사 MMSC URL 하드코딩 (TelephonyManager.networkOperator로 식별)
- 상세 문서: `docs/MMS_LMS_장문문자_발송_구현가이드.md`

### 이미지/파일 첨부
- 이미지: MMS PDU에 image/jpeg 파트 추가 → MMSC HTTP POST
- 다중 이미지: 280KB를 이미지 수로 분배 압축
- 파일: Firebase Storage 업로드 → is.gd URL 단축 → SMS로 링크 발송

### UI 기능
- 메시지 롱프레스: 텍스트 선택 + 클립보드 복사
- URL 자동 감지: 파란색 밑줄 + 탭하면 브라우저 열기
- 전화 아이콘: ACTION_DIAL 인텐트 연동
- 카테고리: 대화 목록 필터 칩 + 롱프레스 배정 + 연락처 탭 배정
- 새 메시지: 이름 검색/번호 입력 분리, 1건→대화창 이동, 다건→완료 토스트
- 메시지 즉시 표시: DB 먼저 저장 → 발송은 백그라운드

### 중복 발송 해결
- SmsSender: 시스템 provider 저장 시 systemSmsId 가져와서 Room에도 저장
- SmsSyncManager: incremental sync 시 systemSmsId로 중복 필터링

### network_security_config.xml
- MMSC 도메인만 HTTP 허용 (LG U+, SKT, KT)
- 나머지는 HTTPS 유지
- `usesCleartextTraffic` 제거 → `networkSecurityConfig`으로 교체
