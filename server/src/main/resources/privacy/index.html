개인정보처리방침

(주)다인온 · BizConnect
시행일: 2026년 3월 22일

(주)다인온(이하 "회사")은 「개인정보 보호법」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 다음과 같이 개인정보처리방침을 수립하여 공개합니다.

제2조 (개인정보의 수집 방법)

회사는 다음과 같은 방법으로 개인정보를 수집합니다.

  • 회원가입 시 사용자가 직접 입력
  • SMS 본인인증을 통한 전화번호 확인
  • 사용자가 명시적으로 동의한 경우에만 주소록 및 SMS 이력 동기화
  • 서비스 이용 과정에서 자동 생성되는 정보

제3조 (개인정보의 이용 목적)

  • 회원 관리: 가입, 본인확인, 부정이용 방지
  • 서비스 제공: SMS/LMS/MMS 발송, AI 문자 작성 지원, 고객 관리
  • 자동 콜백: 통화 종료·부재중 전화번호로 회신 문자 자동 발송 (통화기록은 회신 대상 번호 확인 용도로 기기 내에서만 사용하며, 서버로 전송하지 않습니다)
  • 서비스 개선: 이용 통계 분석, 서비스 품질 향상
  • 보안: 비정상 접근 감지, 기기 인증

항목보유 기간
회원 정보회원 탈퇴 시까지

제5조 (개인정보의 제3자 제공)

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우에는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우
  • SMS 발송을 위해 통신사(세종텔레콤/와이드샷)에 수신자 전화번호 전달

제6조 (개인정보의 안전성 확보 조치)

  • 비밀번호 BCrypt 해시 암호화
  • JWT 토큰 기반 인증
  • SSL/TLS 암호화 통신
  • 새 기기 로그인 시 SMS 2차 인증
  • 10분 비활동 시 자동 로그아웃 (웹)
  • Rate Limiting으로 무차별 공격 방지

제7조 (이용자의 권리와 의무)

이용자는 언제든지 다음을 할 수 있습니다.

  • 개인정보 열람, 수정, 삭제 요청
  • 회원 탈퇴 및 계정·데이터 삭제 (제7조의2 참조)
  • 주소록 및 SMS 이력 동기화 동의 철회
  • 인증된 기기 해제

제7조의2 (회원 탈퇴 및 계정·데이터 삭제)

이용자는 다음 방법으로 계정과 수집된 개인정보의 삭제를 요청할 수 있습니다.

  • 앱에서 직접 삭제: BizConnect 앱 → 설정 → 회원 탈퇴 → 이메일·비밀번호 확인 후 즉시 처리
  • 이메일 요청: 앱 설치 없이도 pcon@naver.com 으로 가입 이메일과 함께 삭제를 요청하시면 처리해 드립니다

회원 탈퇴 시 계정 정보, 문자 발송/수신 이력, 주소록, 발송 이력, 결제 잔액, 인증 기기 정보 등 서버에 저장된 개인정보는 즉시 영구 삭제되며 복구할 수 없습니다. 단, 전자상거래법 등 관련 법령에 따라 보존 의무가 있는 결제·거래 기록은 해당 법정 기간 동안 분리 보관 후 파기합니다(제4조 참조).

제8조 (광고성 문자 발송에 관한 사항)

이용자는 광고성 문자 발송 시 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수해야 합니다.

  • 광고성 문자는 반드시 유료 API를 통해 발송해야 합니다
  • 폰 문자로 광고성 문자를 발송하는 것은 법적으로 금지됩니다
  • 광고성 문자 발송 시 (광고) 표기, 080 수신거부번호 포함, 야간(21~08시) 발송 금지를 준수해야 합니다
  • 이를 위반하여 발생하는 모든 법적 책임은 전적으로 이용자에게 있습니다
