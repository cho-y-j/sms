Features

비즈니스에 필요한 모든 것

문자 앱 하나로 고객 관리부터 마케팅까지. 복잡한 건 AI에게 맡기세요.

💬

기본 SMS 앱

삼성 메시지 수준의 완성도. 기본 앱으로 설정하면 모든 문자를 BizConnect에서 관리합니다.

🚀

대량 발송

핸드폰 직접 발송(무료) + 웹 API 발송(유료). 변수 치환(%이름%)으로 개인화된 대량 메시지.

🧠

AI 어시스턴트

대화 요약, 답장 추천, 감정 분석, 톤 변환, 약속 자동 감지. BizConnect AI가 도와드립니다.

⏰

예약 발송

원하는 시간에 자동 발송. 매일/매주/매월 반복 설정으로 정기 안내 문자도 간편하게.

📞

자동 콜백

통화 종료 후 자동으로 문자 발송. 부재중/수신/발신 별도 설정. 명함 이미지 첨부.

👥

고객 CRM

카테고리별 분류, 메모, 발송 이력. 연락처와 통합된 가벼운 CRM.

🛡

스팸 차단

키워드/번호 차단, 해외번호 필터링, AI 스팸 판별. APick DB 연동으로 정확한 차단.

📋

템플릿

자주 쓰는 문구를 저장하고 변수를 자동 치환. 이미지 포함 MMS 템플릿도 지원.

💳

간편 결제

NICE Pay 연동. 카드/카카오페이/네이버페이로 크레딧 충전 후 유료 발송.
